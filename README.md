
## 1. 게임 개요
* **게임명:** 구구단 갤러그: 마스터 에디션 (Math Galaga: Multiplier)
* **장르:** 에듀테인먼트 (아케이드 슈팅 + 수학 학습)
* **플랫폼:** 반응형 웹앱 (PC, 스마트폰, 태블릿, 대형 전자칠판 지원)
* **핵심 컨셉:** 추억의 레트로 아케이드 게임 '갤러그'의 감성에 구구단 학습을 결합하여, 몰입감 높고 박진감 넘치는 두뇌 플레이를 유도하는 교육용 멀티플레이 게임입니다.

## 2. 핵심 특징 (Key Features)

### 👥 최대 4인 동시 플레이 (경쟁 모드)
* 화면 분할(Lane) 방식을 적용하여 1인용부터 최대 4인용까지 지원합니다.
* 대형 전자칠판에 띄워두고 여러 학생이 동시에 화면을 터치하며 암산 대결을 펼칠 수 있어 교실 내 참여도를 극대화합니다.

### 📈 동적 난이도 조절 시스템 (Leveling)
* 플레이어의 점수가 500점 오를 때마다 레벨업이 발생합니다.
* 레벨이 오를수록 적군의 하강 속도가 가속되고 등장 주기(스폰 레이트)가 짧아져 긴장감을 유발합니다.
* 레벨업에 따라 등장하는 외계 우주선의 형태가 변화(👾 → 👽 → 🛸 → 🤖 → 👿)하여 시각적인 성취감을 제공합니다.

### 🕹️ 완벽한 반응형 레이아웃 및 UX 최적화
* 모바일 환경(1인용)에서는 화면 중앙 정렬 및 키패드 크기 제한을 통해 한 손 조작의 편의성을 높였습니다.
* 전자칠판이나 대형 모니터에서는 다중 레이아웃이 화면 잘림 없이 꽉 차게 렌더링되며, 다중 터치 시 발생할 수 있는 드래그 현상 및 터치 딜레이를 원천 차단했습니다.

### 🎧 8-bit 레트로 감성
* 별도의 외부 이미지나 음원 파일 없이, 브라우저 내장 웹 오디오 API(Web Audio API)를 활용하여 정답(고음), 오답(저음), 레벨업(상승음) 등의 레트로 효과음을 자체 합성하여 출력합니다.
* 픽셀 폰트, CRT 모니터 스타일의 네온 글로우(Glow) 효과, CSS 기반의 미사일 및 폭발 애니메이션을 통해 고전 게임의 향수를 완벽히 재현했습니다.

## 3. 게임 플레이 로직
1.  **문제 출제:** 화면 상단에서 구구단 문제(예: `7 x 8`)를 단 외계 우주선이 무작위로 생성되어 하강합니다.
2.  **정답 입력:** 하단의 숫자 키패드를 이용해 정답을 입력하고 `FIRE` 버튼을 누릅니다.
3.  **격추 및 득점:** 정답이 일치하면 플레이어의 우주선에서 미사일이 발사되어 적을 격추(`💥`)하고 100점을 획득합니다.
4.  **생명 차감 (패널티):** 오답을 입력하거나, 적 우주선이 플레이어의 수비 라인(바닥)까지 닿게 되면 생명(🚀)이 1개 차감되며 화면이 붉게 점멸합니다.
5.  **게임 오버:** 3개의 생명을 모두 잃으면 해당 플레이어의 게임이 종료되며 최종 점수가 표시됩니다.

## 4. 기대 효과 (교육적 가치)
* **연산 능력 향상:** 시간제한(적의 하강)이라는 압박감 속에서 정답을 도출해야 하므로, 단순 암기를 넘어선 **직관적인 빠른 연산(Mental Math)** 능력을 기를 수 있습니다.
* **학습 스트레스 완화:** 지루할 수 있는 수학 연산을 슈팅 게임의 타격감과 레벨업 시스템으로 풀어내어, 학생들이 놀이하듯 자연스럽게 수학에 접근하게 합니다.
* **협동 및 선의의 경쟁:** 칠판 앞에서 친구들과 나란히 서서 각자의 화면을 터치하며 대결하는 과정은 교실 분위기를 활기차게 만듭니다.
